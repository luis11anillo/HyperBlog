complementos de comando CMD:


git status                ----------> revisamos el estado del git en la carpeta 
git add name.txt          ----------> agg el Archivo a git 
git rm --cached name.txt  ----------> sacamos el Archivo del estado de agregado a git 
git commit  -m "message"  ----------> para subir el Archivo al repositorio / (-m "message") se utiliza como 
                                      buena practica 
git log name.txt          ----------> muestra el historial de los cambios realizados y quien los realizo
git show name.txt         ----------> muestra exactamente que fue lo que se cambio

git diff commit_codigo1 commit_codigo2  -------> para comparar los cambios realizados
